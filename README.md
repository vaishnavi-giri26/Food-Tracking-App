🍔 Food Tracking App (Full Stack)

A full-stack food ordering and tracking application with user authentication, cart management, and admin order control.

---

🚀 Live Demo

- 🌐 Frontend:https://food-tracking-app-qyhd.onrender.com
- ⚙️ Backend: https://food-tracking-backend.onrender.com

---

✨ Features

👤 User (Customer)

- Register & Login authentication
- Browse food menu
- Add items to cart
- Place orders (checkout)
- View order history
- Track order status (Pending → Processing → Delivered)

---

👑 Admin

- Login as admin
- View all customer orders
- Update order status
- Manage order flow

---

🛠️ Tech Stack

Frontend

- React.js
- React Router
- CSS 

Backend

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt.js

⚙️ Installation (Local Setup)

1️⃣ Clone Repo

git clone https://github.com/vaishnavi-giri26/Food-Tracking-App.git
cd Food-Tracking-App

---

2️⃣ Backend Setup

cd backend
npm install
npm start

---

3️⃣ Frontend Setup

cd frontend
npm install
npm run dev
